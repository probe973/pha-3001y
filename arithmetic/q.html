Arithmetic Skills

Instructions

  • Calculators of any description are not allowed.
  • All answers should be written to 2 decimal places (unless requested otherwise).
  • All answers must include units, where appropriate. Answers without units (or incorrect units) will be marked as incorrect.
  • If using scientific notation, use as a number between 1 & 10 e.g. $1.6 \times 10^3$
